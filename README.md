## Overview

TouchDesigner Professional Suite is a complete interactive media platform with multi-display output, network collaboration, and commercial licensing.

Multi-display output manages arrays of projectors, LED panels, and displays with edge blending, warping, and mapping for immersive environments.

Network collaboration synchronizes multiple TouchDesigner instances across networked computers for large-scale installations.

Python scripting extends node functionality with custom logic, data processing, and external system integration.

TouchDesigner Professional Suite is the interactive media platform for professionals who create large-scale installations and live experiences with the multi-display, networking, and scripting tools that commercial interactive projects demand.

## Performance & Integration

TouchDesigner is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows

## System Requirements

- macOS 12 Monterey or later
- Apple Silicon or Intel — Universal Binary
- Works with macOS Ventura, Sonoma and Sequoia
